Установка через официальный репозиторий — рекомендуется

Подключив репозиторий один раз, дальше получаете обновления стандартной командой apt upgrade / dnf upgrade. Репозиторий хостится на GitHub Pages и не блокируется в РФ.

Debian / Ubuntu / Mint / Pop!_OS

sudo install -d -m 0755 /etc/apt/keyrings
curl -fsSL https://phinik-dev.github.io/ChatVPN-releases/chatvpn.gpg \
  | sudo gpg --dearmor -o /etc/apt/keyrings/chatvpn.gpg
echo "deb [signed-by=/etc/apt/keyrings/chatvpn.gpg] https://phinik-dev.github.io/ChatVPN-releases/apt stable main" \
  | sudo tee /etc/apt/sources.list.d/chatvpn.list
sudo apt update
sudo apt install chatvpn

Fedora / RHEL / Rocky / Alma / openSUSE

sudo curl -fsSL https://phinik-dev.github.io/ChatVPN-releases/rpm/chatvpn.repo \
  -o /etc/yum.repos.d/chatvpn.repo
sudo dnf install chatvpn

На openSUSE используйте sudo zypper install chatvpn. При первой установке dnf/zypper попросит подтвердить импорт GPG-ключа с fingerprint 209334197FCDB901B1E1365CCA2A478FE9E7E4EA — соглашайтесь.

Для Arch / Manjaro / Steam Deck / Bazzite / airgapped систем репозиторий не подходит — смотрите ниже разделы про прямое скачивание.

Прямое скачивание .deb / .rpm / .tar.gz

Все пакеты этого релиза опубликованы в GitHub Release v2.8.2 — скачайте файл под свою архитектуру и используйте одну из команд ниже.

Debian / Ubuntu — .deb

amd64 (обычные ПК и ноутбуки):

sudo apt install ./chatvpn_2.8.2_amd64.deb

arm64 (Raspberry Pi, ARM-устройства):

sudo apt install ./chatvpn_2.8.2_arm64.deb

Fedora / RHEL / Rocky / Alma / openSUSE — .rpm

x86_64 (обычные ПК):

sudo dnf install ./chatvpn-2.8.2-1.x86_64.rpm

aarch64 (ARM):

sudo dnf install ./chatvpn-2.8.2-1.aarch64.rpm

На openSUSE замените dnf на zypper.

Arch / Manjaro / EndeavourOS и другие — .tar.gz

amd64:

tar xzf chatvpn-2.8.2-linux-amd64.tar.gz
cd chatvpn-2.8.2-linux-amd64
sudo ./install.sh

arm64:

tar xzf chatvpn-2.8.2-linux-arm64.tar.gz
cd chatvpn-2.8.2-linux-arm64
sudo ./install.sh

Особые случаи

Bazzite / Fedora Silverblue / Kinoite / Bluefin (rpm-ostree)

Скачайте .rpm-файл и используйте rpm-ostree install — после установки нужна перезагрузка:

sudo rpm-ostree install ./chatvpn-2.8.2-1.x86_64.rpm
sudo systemctl reboot

Steam Deck (SteamOS)

Переключитесь в Desktop Mode, временно отключите readonly-режим файловой системы и установите из .tar.gz: 

sudo steamos-readonly disable
tar xzf chatvpn-2.8.2-linux-amd64.tar.gz
cd chatvpn-2.8.2-linux-amd64
sudo ./install.sh

Проверка подписи

Все пакеты и метаданные репозитория подписаны GPG-ключом ChatVPN с fingerprint: 

209334197FCDB901B1E1365CCA2A478FE9E7E4EA

Публичный ключ — https://phinik-dev.github.io/ChatVPN-releases/chatvpn.gpg. При установке через apt/dnf подпись проверяется автоматически.
